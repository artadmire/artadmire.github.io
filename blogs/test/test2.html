    1. Vue中 Runtime + Compiler 和 Runtime-only 两种模式含义和区别
    • runtime-only 比 runtime-compiler 轻 6kb，代码量更少
    • runtime-only 运行更快，性能更好
    • runtime-only 其实只能识别render函数，不能识别template，.vue 文件中的template也是被vue-template-compiler 翻译成了render函数，所以只能在.vue里写 template
    • Runtime + Compiler可以识别template方式写代码

    两者的编译流程：

    • runtime-only render -> vdome -> 真实dom
    • Runtime + Compiler template -> render -> vdom -> 真实dom

    建议在生产环境环境中使用runtime-only。
