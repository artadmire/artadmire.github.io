    # 浏览器请求

    # XMLHttpRequest

    # 特点

    1. 所有浏览器都支持 onreadystatechange

    2. 更多现代浏览器，除了可以设置 on* 属性外，也可以用 addEventListener来监听XMLHttpRequest 事件。

    3. 可以终止

    4. 支持二进制

    5. 可以使用FormData对象管理表单数据。

    # 基本用法

    function sendAjax() {
  //构造表单数据
  var formData = new FormData();
  formData.append('username', 'johndoe');
  formData.append('id', 123456);
  //创建xhr对象 
  var xhr = new XMLHttpRequest();
  // 设置请求头
  xhr.withCredentials = true // 允许跨域
  //设置xhr请求的超时时间
  xhr.timeout = 3000;
  //设置响应返回的数据格式
  xhr.responseType = "text";
  //创建一个 post 请求，第三个参数是ture 表示采用异步
  xhr.open('POST', '/login', true); 
  xhr.setRequestHeader('X-Test', 'one');

  //注册相关事件回调处理函数
  xhr.onload = function(e) {
    if(this.readyState === 4) {
      if(this.status == 200 || this.status == 304){
        alert(this.responseText);
      }
    }
  };
  xhr.ontimeout = function(e) {  };
  xhr.onerror = function(e) { };
  xhr.abort = function(e) { }; 
  xhr.upload.onprogress = function(e) {
    if (e.lengthComputable) {
      // 获取
      var completedPercent = e.loaded / e.total;
    }
  };
  
  //发送数据
  xhr.send(formData);

    # 跨域资源共享

    CORS 是一个 W3C 标准，全称是“跨域资源共享”（Cross-origin resource sharing）。它允许浏览器向跨域的服务器，发出XMLHttpRequest请求，从而克服了 AJAX 只能同源使用的限制。

    浏览器发现这次跨域 AJAX 请求，会在请求头中带origin字段。

    GET /cors HTTP/1.1
Origin: http://api.bob.com
Host: api.alice.com
Accept-Language: en-US
Connection: keep-alive
User-Agent: Mozilla/5.0...

    # 如何获取上传、下载的进度

    在上传或者下载比较大的文件时，实时显示当前的上传、下载进度是很普遍的产品需求。

    我们可以通过onprogress事件来实时显示进度，默认情况下这个事件每50ms触发一次。需要注意的是，上传 过程和下载过程触发的是不同对象的onprogress事件：

    • 上传触发的是xhr.upload对象的 onprogress事件

    • 下载触发的是xhr对象的onprogress事件

    xhr.onprogress = updateProgress;
xhr.upload.onprogress = updateProgress;
function updateProgress(event) {
    if (event.lengthComputable) {
      var completedPercent = event.loaded / event.total;
    }
 }

    # 需要取消的场景

    • 当我们从一个页面跳转到另外一个页面时，如果前一个页面的请求还没有返回，那么我们希望取消前一个页面的请求。
    • 某些操作耗时比较长（不能是保存等操作哦），如果用户不想等待呢，取消了操作，对应我们也需要取消HTTP请求。

    # 取消请求

    var xhr = new XMLHttpRequest();
xhr.open('get', 'https://artadmire.github.io', true);
xhr.send();
xhr.onreadystatechange= function (){
console.log(xhr.responseText, '-- respone')
}
// 需要取消的时候调用
setTimeout(() => {xhr.abort()}, 20);

    # axios

    # 特点

    1. 基于xhr
    2. 基于promise
    3. 支持node
    4. 请求拦截
    5. 自动转换JSON数据
    6. 防御xsrf(跨站域请求伪造)

    # 基本用法

    查看：Axions中文使用说明 (opens new window)

    # 取消请求

    var source = axios.CancelToken.source();

axios.get('https://artadmire.github.io', {
  cancelToken: source.token
}).catch(function(err) {
  if (axios.isCancel(err)) {
    // handle our cancel operation
    console.log('Request canceled', err.message);
  } else {
    // handle real error here
  }
});


// 需要取消请求时，调用：
source.cancel('Abort Request');

    # fetch

    # 特点

    1. Fetch是浏览器原生的 api
    2. 返回promise
    3. 仅当网络故障时或请求被阻止时，才会标记为 reject。
    4. 默认fetch 不会发送 cookies
    5. 有兼容性，IE用不了

    # 基本用法

    
fetch('https://artadmire.github.io/')
  .then(response => response.json())
  .then(json => console.log(json))
  .catch(err => console.log('Request Failed', err));

  res.set('Access-Control-Allow-Headers', 'content-type')
  res.set('Access-Control-Allow-Credentials', 'true')
  res.set('Access-Control-Allow-Methods', 'PUT,POST')
  res.send('app2')

    # 简单请求 & 非简单请求

    简单请求 （1) 请求方法是以下三种方法之一：

    • HEAD

    • GET

    • POST （2）HTTP的头信息不超出以下几种字段：

    • Accept

    • Accept-Language

    • Content-Language

    • Last-Event-ID

    • Content-Type：只限于三个值application/x-www-form-urlencoded、multipart/form-data、text/plain

    简单请求实现跨域,下面的设置就可以满足：

    app.get('/', function (req, res) {
  res.set('Access-Control-Allow-Origin', 'http://localhost:3001')
  res.set('Access-Control-Allow-Credentials', 'true')
  res.send('hello app2')
})

    非简单请求

    非简单请求是那种对服务器有特殊要求的请求，比如请求方法是PUT或DELETE，或者Content-Type字段的类型是application/json。

    fetch('http://localhost:3001/api',
{

  method: 'POST'
  headers: {'Content-Type': 'application/json'},
  data: JSON.stringify({a: 1})

    # 判断请求成功

    async function fetchText() {
  let response = await fetch('/readme.txt');
  if (response.status >= 200 && response.status < 300) {
    return await response.text();
  } else {
    throw new Error(response.statusText);
  }
}

    # 取消请求

    fetch与XMLHttpRequest(XHR)类似，是ES6之后浏览器（除IE之外）默认支持的http操作函数。默认不支持abort操作。可以通过AbortController来实现。

    代码：

    // 声明AbortController
const controller = new AbortController();
const signal = controller.signal;
// 正常的http调用
fetch('http://localhost:3000', { signal })
    .then(r => r.json())
    .then(response => {
        console.log(response);
    })
    .catch(err => {
        if (err.name === 'AbortError') {
            console.log('Fetch was aborted')
        } else {
            console.log('Error', err)
        }
    });

signal.addEventListener('abort',
  () => console.log('abort!')
);

// 需要取消请求时，调用：
controller.abort()
