    # Web Components

    If I have written this article it is by standing on the shoulders of giants. Pay tribute!

    # 什么是Web Components？

    可以理解为，在不使用框架的情况下，能在html中使用自定义标签，并实现传递参数，绑定事件的技术。

    这样就可以实现重复使用，降低耦合的效果。

    # customElements

    定义一个自定义标签，先来认识一下customElements api。这个api可以让你通过下面的方式定义一个自定义标签：

    customElements.define('word-count', WordCount, { extends: 'p' });

    class WordCount extends HTMLParagraphElement {
  constructor() {
    // 必须首先调用 super 方法
    super();

    但是这样会引起其它的问题，样式相互污染，如果说页面中也有一个相同class名字的元素，样式也会变成红色，这就违背了松耦合的初衷。

    # Shadow DOM

    为了不让 template 里的 style标签中 CSS 和全局的 CSS 有冲突，我们可以将组件挂在到 Shadow Root 上，再用 Shadow Root 挂到外层的 document DOM 上，这样就可以实现 CSS 的隔离啦：

       renderV1() {
        this.attachShadow({ mode: 'open' })
        const tem = document.getElementById('web-button-template');
        const clonedElem = tem.content.cloneNode(true);
        this.shadowRoot.appendChild(clonedElem)
    }

    # 添加事件

    这个也比较简单，直接获取到相应元素添加事件就可以。

        renderV1() {
        this.attachShadow({ mode: 'open' })
        const tem = document.getElementById('web-button-template');
        const clonedElem = tem.content.cloneNode(true);
        clonedElem.querySelector('.btn').addEventListener('click', this.click)
        this.shadowRoot.appendChild(clonedElem)
    }
    click() {
        console.log('btn click')
    }

    # 支持slot

    HTML 模板的另一个好处是可以像 Vue 一样使用 slot。比如，现在我们可以在这个 web-button 最下面添加一个 icon 插槽：

        <template id="web-button-template">
        <style>
          .btn {} /*省略样式*/
        </style>
        <button class="btn">
            提交
            <slot name="icon"></slot>
        </button>
    </template>
    <web-button data-type="info">
        <span slot="icon" >&spades;</span>
    </web-button>
